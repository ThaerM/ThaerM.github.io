The Abu Dhabi Housing Authority (ADHA) application allows Abu Dhabi citizens to apply for housing services digitally using their smartphones or tablets. In addition to providing detailed information about the services, the eligibility calculator helps determine the services the user is eligible for. The applications status function was also updated and loan status has its own section on the home screen. The application supports both Arabic and English languages. The new release contains the new features and services below:

  • · A new section for loan status was added to the home screen to access its details easily.
  • · Mustasharak service allows the user to contact a consultant from ADHA through booking an appointment or submitting an e-consultation request.
  • · Qasemati service provides the users with their land details and location on the map.
  • · Loan calculator allows the user to calculate the loan amount and the monthly repayments.
